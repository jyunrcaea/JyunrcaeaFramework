# Jyunrcaea! Framework
SDL2를 이용하여 제작한 2D 앱/게임 개발용 프레임워크.

## Debuging
이 프로젝트는 라이브러리 형식을 갖추고 있습니다.<br>
다음 두가지 방법을 통해 디버깅을 시도해볼수 있습니다.

### Add startup project
(Visual Studio 2022 기준)<br>
솔루션에 콘솔 프로젝트를 추가하고 시작 프로젝트로 설정한뒤, 콘솔 프로젝트의 종속성에서 Jyunrcaea! Framework 프로젝트를 참조하세요.

### Add new file.
프로젝트를 콘솔 프로젝트 유형으로 바꾸고 (Visual Studio 2022 기준)<br>
원하는 이름의 파일을 추가하거나 'JyunrcaeaFramework.cs' 파일 내에 진입점을 생성합니다.

### Start debuging
이후 콘솔 프로젝트에서 Jyunrcaea! Framework 가 제공하는 명령어를 사용하여 디버깅을 해볼수 있습니다.
